HTML Works

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript, both of which will be covered in following sections.

This page will be used to demonstrate a comprehensive understanding of HTML elements as introduced in the The Complete Web Developer Course 2.0hosted on Udemy.com

All pages in this section only use basic HTML elements, except for the nav bar, which is from bootstrap. The final page, an info page about the Earth's moon, shows all of these elements as what most would be consider a complete webpage.
